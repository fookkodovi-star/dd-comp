### 2. API компонента (TreeProps) ✅

Все обязательные опции реализованы:

- ✅ `data: TreeDataNode[]` - дерево узлов
- ✅ `renderLabel?: (node) => ReactNode` - полная свобода отрисовки
- ✅ `getChildren?: (node) => TreeDataNode[]` - кастомный геттер (для lazy loading)
- ✅ `defaultExpanded?: string[]` - неконтролируемое состояние
- ✅ `expanded?: string[]` + `onExpandedItemsChange?` - контролируемое состояние
- ✅ `defaultSelectedItems?: string | string[]` - неконтролируемое состояние
- ✅ `selected?: string | string[]` + `onSelectedItemsChange?` - контролируемое состояние
- ✅ `multiSelect?: boolean` - множественный выбор
- ✅ `nodeContentSx?: (node) => SxProps` - пер-узловой override стилей
- ✅ `sx?: SxProps` - глобальный стиль контейнера
- ✅ Все пропсы типизированы в `types.ts`

## 🎯 Ключевые особенности реализации

### Типизация (types.ts)

```typescript
// Полная типизация без any
interface TreeDataNode<T extends Record<string, unknown>>
interface NodeMeta
interface TreeProps<T extends Record<string, unknown>>
type NodeStatus = 'active' | 'stopped' | 'conflict' | 'partial'
```

### Утилиты (utils.ts)

- `hasChildren()` - проверка наличия детей
- `defaultGetChildren()` - дефолтный геттер
- `mergeSx()` - объединение стилей
- `getStatusColor()` - цвет по типу статуса
- `getAllNodeIds()` - получение всех ID

### Стили (styles.ts)

- `treeSx` - базовые стили контейнера
- `treeItemContentBaseSx` - стили карточки узла
- `labelSx`, `labelTextSx` - стили текста
- `metaContainerSx` - стили метаданных
- `statusContainerSx`, `statusIndicatorSx` - стили статусов
- `highlightLeftBorder()` - функция для выделения

### CustomTreeItem.tsx

- Мемоизация с React.memo
- Рекурсивная отрисовка children
- Инкапсуляция slots и slotProps
- Transition через Collapse
- DefaultLabel с полной поддержкой метаданных

### Tree.tsx

- Оболочка над SimpleTreeView
- Управление контролируемым/неконтролируемым состоянием
- Проксирование всех пропсов
- Глобальные иконки и стили
